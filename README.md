# 🦁 Genesis Savanna Simulation

**Genesis物理エンジン上で動くサバンナ生態系シミュレーター。**  
強化学習 (REINFORCE) × 進化的アルゴリズム (DNA進化) を組み合わせ、動物たちが世代を重ねるごとに生き残る術を学んでいく様子を3Dビジュアルで観察できます。


[![](https://img.youtube.com/vi/gIsO6bSz6hk/0.jpg)](https://youtu.be/gIsO6bSz6hk?si=Vhba91YJyOdEeEee)

## 📺 概要

| 項目 | 内容 |
|------|------|
| 物理エンジン | [Genesis](https://github.com/Genesis-Embodied-AI/Genesis) |
| 学習手法 | REINFORCE (Policy Gradient) |
| 進化手法 | 遺伝的アルゴリズム (一様交叉 + ガウス突然変異) |
| 出力 | ONNX ポリシーファイル × 4種 + `evolution_history.json` |

### デモの見どころ

- 🌅 **昼夜サイクル** — 1エピソード = 1日。日の出から日没まで太陽が東から西へ動き、空の色・影の角度がリアルタイムに変化
- 🧬 **DNA進化** — 世代をまたいで体質・行動傾向が自然選択される。世代ログで遺伝子の変化を追える
- 🗺️ **立体地形** — 山(登頂可能なのはライオンのみ)・森林(草食獣の隠れ場)・水飲み場・平地の4種
- 🤖 **種ごとに独立したRL** — ライオンと草食獣でポリシーネットワークが分かれており、捕食・逃走の拮抗が生まれる

## 🌍 世界の構造

```
ワールドサイズ: 20 × 20 グリッド (CELL = 1.0 m)
1エピソード  : 最大 300 ステップ = 1日
```

### 地形タイプ

| 地形 | 見た目 | 特性 |
|------|--------|------|
| 平地 | 砂色ベース | 全動物通行可。草食獣が採食できる |
| 山 🏔 | 灰褐色の立体Box + 山頂岩 | **ライオンのみ**通行可 |
| 森林 🌳 | 幹 + 樹冠の2層構造 | ライオンが近いとき草食獣に隠れボーナス |
| 水飲み場 💧 | 薄い青いBox | 全動物の渇きを回復。`water_sense`が高いほど回復量UP |

## 🐾 動物

| 種 | 頭数(デフォルト) | 役割 | 特徴 |
|----|----------------|------|------|
| ライオン 🦁 | 3 | 捕食者 | 山を越えられる。捕食で空腹-50%・報酬+10 |
| シマウマ 🦓 | 8 | 草食獣 | バランス型。群れ志向が高い |
| ガゼル 🦌 | 10 | 草食獣 | 最速・視野最大。ライオンから最も敏感に逃げる |
| 象 🐘 | 4 | 草食獣 | 低速・大型。水をよく飲む (`thirst_rate` が高め) |

### 生死のルール

```
毎ステップ:
  thirst += thirst_rate   (水場滞在で -0.25〜0.50 回復)
  hunger += hunger_rate   (草地・森で -0.05 回復 / 捕食成功で -0.50)

死亡条件:
  thirst >= 1.0  → 脱水死
  hunger >= 1.0  → 飢餓死
  ライオンとの距離 < 0.6 + size_scale×0.25  → 捕食死 (即死)
```

## 🧬 DNA進化システム

各個体は **7つの遺伝子** を持ち、世代をまたいで進化します。

### 遺伝子の種類

| 遺伝子 | 範囲 | 何に影響するか |
|--------|------|----------------|
| `speed` | 0.5 〜 2.5 | 1ステップあたりの移動距離 |
| `thirst_rate` | 0.002 〜 0.012 | 渇きの増加速度 |
| `hunger_rate` | 0.002 〜 0.010 | 空腹の増加速度 |
| `size_scale` | 0.6 〜 1.5 | 体の大きさ。ライオンは捕食半径も変化 |
| `boldness` | 0.0 〜 1.0 | ライオン接近時のペナルティ感度 (低=臆病) |
| `sociality` | 0.0 〜 1.0 | 群れ志向 (観測ベクトルに渡される) |
| `water_sense` | 0.0 〜 1.0 | 水場での回復量と報酬ボーナスに影響 |

### 世代交代フロー

```
1世代 = EPISODES_PER_GEN エピソードを走らせて fitness を蓄積
         ↓
fitness = 生存ステップ数 + 捕食数×15 - (渇き+空腹)×10
         ↓
上位 ELITE_RATIO (40%) を親として保存 (エリート保存)
         ↓
一様交叉 + ガウス突然変異 (rate=15%, σ=0.12) で次世代を生成
         ↓
N_GENERATIONS 世代繰り返す
```

## 🤖 強化学習 (REINFORCE)

DNAは「体質」を決め、ポリシー (ニューラルネット) は「その体質の中での最適な動き方」を学びます。

### 観測ベクトル (16次元)

```
[0,1]   自分の位置 (x/20, y/20)
[2]     現在の地形タイプ (0〜1)
[3,4]   渇き・空腹 (0〜1)
[5,6,7] 最近傍水場への方向 (dx, dy) と距離
[8,9,10] 脅威 (ライオン/草食獣) への方向と距離
[11]    森林近傍フラグ
[12,13] 最近傍同種仲間の方向
[14]    boldness  (DNA遺伝子を直接渡す)
[15]    water_sense
```

### ネットワーク構造

```
Linear(16 → 128) → ReLU → Linear(128 → 128) → ReLU → Linear(128 → 5)
出力: 5行動の logits (北/南/西/東/静止)
```

### 報酬設計

| 条件 | 報酬 |
|------|------|
| 生存コスト (常時) | -0.01 |
| 渇きペナルティ | -thirst × 0.5 |
| 空腹ペナルティ | -hunger × 0.3 |
| 死亡 | -5.0 |
| 草食獣: ライオン接近 | -(4-dist) × (1 + 1-boldness) |
| 草食獣: 森林滞在 | +0.3〜0.8 |
| 草食獣: 水場滞在 | +1.0 + water_sense×1.0 |
| ライオン: 草食獣に接近 | +(3-dist) × 0.5 |
| ライオン: 捕食成功 | +10.0 |

## 🚀 セットアップ

### 必要環境

- Python 3.10 以上
- macOS / Linux (Windows は Genesis の対応状況を確認してください)
- GPU 推奨 (CUDA / Apple MPS 自動検出)

### インストール

```bash
# Genesis
pip install genesis-world

# その他依存
pip install torch onnx onnxruntime numpy
```

### 実行

```bash
# デフォルト設定で実行
python savanna_sim.py

# 個体数を少なめにして動作確認
python savanna_sim.py --lions 2 --zebras 4 --gazelles 5 --elephants 2

# 世代数・エピソード数も指定
python savanna_sim.py --lions 2 --zebras 4 --gazelles 5 --elephants 2 --gens 20 --eps 5

# フルスケール
python savanna_sim.py --lions 3 --zebras 8 --gazelles 10 --elephants 4 --gens 50 --eps 10

# ヘルプ
python savanna_sim.py --help
```

### 停止方法

```bash
# Ctrl+C (シグナルハンドラで即終了)

# 止まらない場合は別ターミナルから
pkill -9 -f savanna_sim.py
```

## 📈 進化の期待動向

世代を重ねると以下の方向に遺伝子が収束していく傾向があります。

- **ライオン** — `speed` ↑、`hunger_rate` ↓ (少ない捕食で長く生き延びる)
- **シマウマ/ガゼル** — `thirst_rate` ↓、`boldness` ↓ (ライオンを避けつつ効率よく水を飲む)
- **象** — `water_sense` ↑ (水場への依存度が高いため感度が強化される)
